# catalog-digital
Un proiect de facultate ce reprezinta un catalog digital capabil sa adauge, sa stearga si sa modifice note in functie de ID.
Aceasta aplicatie are o interfata de tip CLI.
In stagiul curent al proiectului, acesta prezinta bug-uri minore, si trebuie imbunatatit

Pentru functionarea buna a programului, path-ul variabilelor de tip fisier trebuie schimbat in codul sursa (main.java) dupa caz
