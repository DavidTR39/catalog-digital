# catalog-digital
Un proiect de facultate ce reprezinta un catalog digital capabil sa adauge, sa stearga si sa modifice note in functie de ID-ul studentului.
Notele sunt stocate intr-o baza de date intr-un fisierde tip text.
Cand programul porneste, acesta citeste fisierul text pentru a gasi note pre-existente in baza de date.
Aceasta aplicatie are o interfata de tip CLI.
In stagiul curent al proiectului, acesta prezinta bug-uri minore, si trebuie imbunatatit

Pentru functionarea buna a programului, path-ul variabilelor de tip fisier trebuie schimbat in codul sursa (main.java) dupa caz
-------------------------------------------
A college project which represents a digital catalogue cabable of adding, removing and modifying grades based on the student's ID.
The grades are stored in a database from a plain text file.
When the program runs, it reads the text file for any pre-existing grades from the database.
This application has a CLI-type interface
In the current stage of the project, it has minor bugs and must be improved

**For the good functioning of the program, the file variables's path must be changed in the source code (main.java) as appropiate.**
